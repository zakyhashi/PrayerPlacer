Prayer Placer

Prayer Placer is a mobile app designed to help Muslims find safe, quiet, and accessible places to pray in urban environments. It reduces the stress of finding suitable prayer spaces while supporting users in maintaining their daily prayers.

Features
- Live Map: View nearby prayer spaces including mosques and prayer rooms
- Community Contributions: Users can add and share new locations
- Prayer Times: Access daily prayer schedules
- Qibla Direction: Built-in compass for prayer direction
- Settings: Basic customization options

Purpose
Prayer Placer addresses the difficulty of finding safe and comfortable prayer spaces in cities by providing a reliable, community-driven solution.

Tech Stack
- Swift / SwiftUI
- CoreLocation
- Map Integration (planned)
- iOS (Xcode)

How It Works
1. Open the app to view nearby prayer spaces
2. See your location and available options
3. Select a location and get directions
4. Add new spaces to help others
